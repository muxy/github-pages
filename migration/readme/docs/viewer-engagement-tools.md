# Viewer Engagement Tools

Build extensions that let streamers and broadcasters interact with viewers.

Basic aggregation tools let you collect user data and do some statistical analysis on it.  How you use that data is up to you. However, MEDKit also offers tooling that expands on the use of aggregated data, helping you to set up activities that encourage and reward user participation.

Special-purpose tools let you set up questions or prompts to elicit different kinds of responses from viewers, encourage interaction between viewers and broadcasters or streamers, and offer incentives that keep viewers actively engaged.

* [Polls and User Voting](https://docs.muxy.io/docs/voting) : This example illustrates how to poll users, collect their votes, and work with voting results.
