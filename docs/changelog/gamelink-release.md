# GameLink Launch Announcement

This is the original public-launch announcement for Muxy's real-time game-integration platform. The legacy export did not include a publication date.

!!! warning "Historical release note"
    The repositories linked by the original announcement are public, but the GameLink C++, C#/Unity, and Unity sample repositories currently publish no supported GitHub release tags. Do not use this announcement to select an SDK version. Use the pinned [Unity tutorial](../docs/unity-gamelink-tutorial.md), [native SDK reference](../reference/gamelink-library.md), or [WebSocket protocol reference](../reference/websocket-protocol.md) and observe their release warnings.

## Original launch scope

The launch introduced:

- GameLink documentation and API references on `docs.muxy.io`.
- Extension registration and metrics in the [Muxy Developer Portal](https://dev.muxy.io).
- Public source for [GameLink C#](https://github.com/muxy/gamelink-csharp) and [GameLink C++](https://github.com/muxy/gamelink-cpp).
- A public [Unity GameLink example](https://github.com/muxy/unity-gamelink-example).
- A public [MEDKit Vue starter](https://github.com/muxy/medkit-starter-vue).

These links are source snapshots, not proof of a currently supported binary or package release. Confirm a tagged release and owner approval before shipping a new GameLink integration.
