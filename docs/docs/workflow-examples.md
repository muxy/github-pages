This section shows some workflows to get you started using Muxy functionality.

[Basic operation examples](../docs/basic-usage-examples.md)  show you how to:

- Broadcast messages to viewers

A more extended example shows you how to [Implement GameLink Polling](../docs/implement-gamelink-polling.md), and [Manage Polling](../docs/manage-polling.md) when you have multiple polls running concurrently.
