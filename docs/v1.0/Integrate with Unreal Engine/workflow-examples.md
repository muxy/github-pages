This section shows some workflows to get you started using Muxy functionality.

[Basic operation examples](../../v1.0/Integrate with Unreal Engine/workflow-examples/basic-usage-examples.md)  show you how to:

- Retrieve and use state information
- Broadcast messages to viewers
- Accept bit transactions

A more extended example shows you how to [Implement GameLink Polling](../../v1.0/Integrate with Unreal Engine/workflow-examples/implement-gamelink-polling.md), and [Manage Polling](../../v1.0/Integrate with Unreal Engine/workflow-examples/manage-polling.md) when you have multiple polls running concurrently.
