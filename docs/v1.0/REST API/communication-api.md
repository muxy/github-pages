The [Broadcast Messaging](../../v1.0/REST API/communication-api/broadcast.md) service allows extensions to communicate.

The `broadcast` service enables a broadcaster or admin to define and publish messages to subscribers.  
A client can broadcast messages to all or some viewers, and also publish and subscribe to message threads.
