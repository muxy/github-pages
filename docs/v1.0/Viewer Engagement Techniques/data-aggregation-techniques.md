This section introduces a variety of techniques for _aggregating viewer data_.  Aggregation is the key to building user-interaction functionality as you scale up to large numbers of viewers. 

Commented example code illustrates the basic aggregation tools and techniques.

- [Accumulating User Data](../../v1.0/Viewer Engagement Techniques/data-aggregation-techniques/accumulation.md) : This example illustrates the simplest form of aggregation.
- [Ranking User Data](../../v1.0/Viewer Engagement Techniques/data-aggregation-techniques/ranking.md) : This example illustrates how to request arbitrary input from viewers and collect responses that are ranked by popularity. 

More sophisticated [Viewer Engagement Tools](../../v1.0/Viewer Engagement Techniques/viewer-engagement-tools.md) build on these basic techniques to help you promote viewer engagement with interactive experiences.
